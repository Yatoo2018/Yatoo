### 短网址

起因：一次写url的检验正则时，了解到短网址对字母大小写敏感，而记忆中网址一般情况不分大小写（上述观点不准确，这里纠正下应该是协议，域名对大小写不敏感，而参数和路劲对大小写是否敏感依赖服务器的环境，window是不敏感的，unix是敏感的。）别跟我一样在记错了哦。
